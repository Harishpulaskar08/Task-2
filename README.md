# Student Record Management System

A simple CLI-based Student Record Management System developed using Java.

## Objective

This project allows users to perform CRUD (Create, Read, Update, Delete) operations on student records through a menu-driven console application.

## Features

- Add Student
- View All Students
- Search Student by ID
- Update Student Details
- Delete Student
- Duplicate ID Validation
- Marks Validation (0–100)
- Exception Handling for Invalid Input

## Technologies Used

- Java
- ArrayList
- Object-Oriented Programming (OOP)

## Project Structure

```
StudentRecordManagement/
│── Student.java
│── StudentManagement.java
│── Main.java
```

## Concepts Covered

- Classes and Objects
- Encapsulation
- Constructors
- ArrayList
- CRUD Operations
- Loops
- Methods
- Exception Handling

## How to Run

1. Clone the repository.
2. Open the project in VS Code or IntelliJ IDEA.
3. Compile all Java files.
4. Run `Main.java`.
5. Use the menu to manage student records.

## Sample Menu

```
1. Add Student
2. View Students
3. Search Student
4. Update Student
5. Delete Student
6. Exit
```
